    How to Get Your TOTP Code

    TOTP means Time-based One-Time Password. It is the 6-digit code generated in your authenticator app every ~30 seconds.

    1. Install an authenticator app: Google Authenticator, Microsoft Authenticator, or Authy.
    2. Click Start MFA Enrollment on the previous screen.
    3. In Microsoft Authenticator, tap Add account -> Other account -> Enter setup key.
    4. Select Time-based account type (not counter-based) and keep phone date/time on automatic sync.
    5. Use your email as account name and paste the Authenticator Secret shown in this portal.
    6. Do not manually type if possible. If typing, treat O as letter O (not zero).
    7. The app will generate a 6-digit code. Enter that in TOTP Code and click Activate MFA.
    8. Save backup codes safely. Each backup code can be used once if your phone is unavailable.

    Security note: never share your secret key, TOTP code, or backup codes.
